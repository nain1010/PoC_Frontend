# Luma Admin Dashboard

Luma es un panel de administración moderno, ligero y de alto rendimiento basado en **React 19** y **TypeScript**. Ha sido diseñado para ser extremadamente rápido, eliminando dependencias innecesarias y optimizando el bundle de CSS para ofrecer una experiencia de usuario fluida.

## 🚀 Características Principales

- **React 19 & TypeScript**: Código tipado y moderno para un desarrollo seguro.
- **Arquitectura Optimizada**: Se han eliminado más de 40 librerías pesadas de la base original para maximizar la velocidad.
- **Diseño Premium**: Interfaz limpia, responsiva y personalizable.
- **Soporte Multilingüe**: Configurado para Español e Inglés.
- **Modos Visuales**: Soporte completo para modo claro (Light) y oscuro (Dark).

## 🛠️ Instalación y Uso

1. **Instalar dependencias**:
   ```bash
   npm install
   ```

2. **Arrancar en desarrollo**:
   ```bash
   npm start
   ```

3. **Construir para producción**:
   ```bash
   npm run build
   ```
